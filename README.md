# Data Science Salaries Analysis (Pandas)

## 📌 Project Overview
This project focuses on analyzing **data science salary data** using **Python and Pandas**.  
The goal is to extract meaningful insights about how salaries vary based on:

- Experience level  
- Job role  
- Remote work status  
- Company size  
- Company and employee location  

This type of analysis is useful for **companies, recruiters, job seekers, and analysts** who want to understand salary trends in the data science industry.

---

## 🎯 Objectives
The main objectives of this project are to:

- Clean and prepare a real-world dataset for analysis  
- Explore salary trends across different job roles and experience levels  
- Analyze the impact of remote work on salaries  
- Compare salaries across countries and company sizes  
- Demonstrate practical Pandas skills used in real client projects  

---

## 📂 Dataset Description
The dataset contains the following columns:

- **work_year** – Year the salary was paid  
- **experience_level** – Level of experience (Entry, Mid, Senior, Executive)  
- **employment_type** – Full-time, Part-time, Contract, Freelance  
- **job_title** – Job role (e.g., Data Scientist, ML Engineer)  
- **salary** – Salary in original currency  
- **salary_currency** – Currency used  
- **salary_in_usd** – Salary converted to USD  
- **employee_residence** – Employee’s country of residence  
- **remote_ratio** – Percentage of remote work (0, 50, 100)  
- **company_location** – Company’s country  
- **company_size** – Company size (Small, Medium, Large)

---

## 📊 Analysis Performed
Key analyses include:

- Average salary by experience level  
- Top-paying data science job roles  
- Salary comparison between remote, hybrid, and onsite roles  
- Salary trends by company size  
- Salary comparison by company location and employee residence  
- Creation of salary bands for better interpretation  

---

## 💡 Key Insights
Some of the insights derived from the analysis include:

- Senior and executive-level roles earn significantly higher salaries than entry-level roles  
- Fully remote roles often offer competitive salaries compared to onsite roles  
- Larger companies generally pay higher average salaries  
- Certain countries consistently offer higher compensation for data science roles  

(Exact values can be found in the analysis notebook.)
